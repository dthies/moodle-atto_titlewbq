atto_titlewbq
=================

This is a plugin for demostration of code and testing the viability of
for backport of code to add blockquote to Moodle Atto title plugin.
To install unpack to lib/editor/atto/plugins/titlewbq, and visit the
admin notifications page to install.  Then add the button on the Atto
toolbar settings page in place to the core title plugin. The indent
plugin may need to be disabled.
